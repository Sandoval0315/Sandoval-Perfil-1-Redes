Mas informacion

Switch: Conecta múltiples dispositivos en una red local (LAN) y envía datos al dispositivo correcto usando direcciones MAC.

Router: Dirige el tráfico de datos entre diferentes redes, como entre una red doméstica y la Internet, utilizando direcciones IP.

Access Point: Permite la conexión inalámbrica de dispositivos a una red, actuando como un puente entre la red cableada y los dispositivos inalámbricos.

Firewall: Protege la red controlando el tráfico de entrada y salida basado en reglas de seguridad predefinidas.

Modem: Convierte las señales digitales en analógicas y viceversa para la transmisión de datos a través de líneas telefónicas o de cable.

Hub: Conecta múltiples dispositivos en una red y envía los datos recibidos a todos los puertos sin distinguir el destino.
